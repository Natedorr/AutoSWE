# Slash Commands — Full Reference

## Navigation & Session

| Command | Purpose |
|---|---|
| `/clear` | Clear terminal + new chat |
| `/new` | Fresh conversation, same CLI session |
| `/resume` | Resume saved conversation |
| `/fork` | Clone conversation to new thread |
| `/side` | Ephemeral side conversation |
| `/quit` / `/exit` | Exit CLI |
| `/copy` | Copy latest output (`Ctrl+O`) |
| `/compact` | Summarize to free tokens |
| `/raw` | Toggle raw scrollback |

## Configuration

| Command | Purpose |
|---|---|
| `/model` | Choose active model |
| `/fast` | Toggle Fast tier |
| `/personality` | Communication style (`friendly`, `pragmatic`, `none`) |
| `/permissions` | Sandbox/approval level |
| `/theme` | Syntax highlighting theme |
| `/keymap` | Remap TUI shortcuts |
| `/vim` | Toggle Vim mode |
| `/statusline` | Configure footer fields |
| `/title` | Configure terminal title fields |
| `/debug-config` | Print config diagnostics |

## Task Control

| Command | Purpose |
|---|---|
| `/plan` | Enter plan mode |
| `/goal` | Set/view/pause/resume/clear task goal |
| `/approve` | Retry denied action |
| `/review` | Code review working tree |
| `/diff` | Show Git diff |
| `/status` | Session config + token usage |
| `/ps` | Background terminals |
| `/stop` | Stop background work |

## Tools & Extensions

| Command | Purpose |
|---|---|
| `/skills` | Browse and select skills |
| `/mcp` | List configured MCP tools |
| `/agent` | Switch subagent thread |
| `/hooks` | Review lifecycle hooks |
| `/apps` | Browse apps (connectors) |
| `/plugins` | Browse installed plugins |

## Context

| Command | Purpose |
|---|---|
| `/mention` | Attach file to conversation |
| `/ide` | Include IDE context |
| `/memories` | Configure memory use |

## Misc

| Command | Purpose |
|---|---|
